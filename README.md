# Java Dungeon RPG 🗡️🐢

Sebuah proyek game Role-Playing Game (RPG) berbasis terminal yang dikembangkan menggunakan bahasa pemrograman Java dengan menerapkan prinsip **Object-Oriented Programming (OOP)** secara mendalam.

## 🌟 Fitur Utama
* **Hero Class System:** Pilih karakter dengan atribut unik.
* **Dynamic Monster Scaling:** Kesulitan musuh meningkat seiring progres permainan.
* **Loot & Inventory:** Sistem drop item dan manajemen perlengkapan.
* **Battle Engine:** Mekanisme pertarungan turn-based yang menantang.

## 🛠️ Arsitektur Kode
Proyek ini menggunakan struktur kelas yang modular untuk memastikan kode mudah dikembangkan:
* `GameEngine.java`: Logika utama perulangan game (game loop).
* `Entity.java`: Base class untuk Hero dan Monster menggunakan konsep inheritance.
* `CombatSystem.java`: Menangani perhitungan damage dan interaksi saat bertarung.
